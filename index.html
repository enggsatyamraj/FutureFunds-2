TOPSIS (📊 Technique for Order Preference by Similarity to Ideal Solution)

Description

TOPSIS is a multi-criteria decision-making method that identifies solutions from a finite set of alternatives based upon their geometric distance to an ideal 🥇 and a nadir 🥈 solution. The chosen alternative should have the shortest distance from the ideal solution and the farthest distance from the nadir solution.

Mathematical Foundation

  1. Normalization: rij=xij∑i=1mxij2

  2. Weighting: vij=wjrij

  3. Determine Ideal and Nadir Solutions: A+={v1+,...,vn+} A−={v1−,...,vn−}

  4. Calculate Separation Measures: Si+=∑j=1n(vij−vj+)2 Si−=∑j=1n(vij−vj−)2

  5. Calculate Relative Closeness to the Ideal Solution: Ci∗=Si−Si++Si−

Merits

  • 💡 Simplicity: Easy to understand and implement.
  • ⚡ Efficiency: Computationally less intensive.
  • 🔄 Versatility: Applicable to various decision-making scenarios.

Entropy Weighting

Description

Entropy Weighting is an objective method to determine the weight of each criterion based on the inherent variability of the data. Higher entropy indicates more disorder and less information, leading to lower weight assignments.

Mathematical Foundation

  1. Calculate Proportion: pij=xij∑i=1mxij

  2. Compute Entropy: ej=−1ln⁡(m)∑i=1mpijln⁡(pij)

  3. Determine Divergence: dj=1−ej

  4. Assign Weights: wj=dj∑j=1ndj

Merits

  • 📊 Objectivity: Relies solely on data variability.
  • 🔄 Adaptability: Automatically adjusts weights based on data distribution.

Application of TOPSIS and Entropy Weighting

Step-by-Step Explanation

  1. Data Preparation:
    • Input: Engineered features for each district and scheme.
    • Objective: Normalize data and assign weights based on entropy.
  2. Entropy Weighting:
    • Calculate Proportions: Determine the relative contribution of each criterion.
    • Compute Entropy: Assess the disorder or uncertainty associated with each criterion.
    • Assign Weights: Allocate weights inversely proportional to entropy, giving higher weights to more informative criteria.
  3. TOPSIS Normalization:
    • Normalize Criteria: Convert data to a comparable scale without distorting differences.
    • Weighted Normalization: Apply the entropy-derived weights to the normalized data.
  4. Determine Ideal and Nadir Solutions:
    • Ideal Solution: Best possible values across all criteria.
    • Nadir Solution: Worst possible values across all criteria.
  5. Calculate Separation Measures:
    • Distance to Ideal: Measure how close each scheme is to the ideal solution.
    • Distance to Nadir: Measure how far each scheme is from the nadir solution.
  6. Compute Relative Closeness:
    • Closeness Coefficient: Calculate the relative closeness to the ideal solution.
    • Ranking: Rank schemes based on the closeness coefficient, with higher values indicating greater suitability.

Entropy Weighting

  2. Entropy Calculation: ej=−1ln⁡(m)∑i=1mpijln⁡(pij)

  3. Divergence and Weights: dj=1−ej wj=dj∑j=1ndj
